My Projects

  • Content Management System

    A system which is used to manage Web Content by various admins. The admins have authorized access to the data which visible to the public as well as the data hidden from the them. The content is being stored in MySQL database using PHP and being displayed in a presentation layer as per the requests to the public as well as the admins.

    The admin information has been encrypted using Blowfish algorithm using salting to add an extra layer of security thus preventing unauthorized access into the system.

  • Ticket World

    Often one wants to look for events happening across the world online. Searching for them at random turns out to be a tedious process. Therefore this project lets you search for events based on a category specified and lists out all the events taking place.

    One can therefore buy tickets from the results that'll direct to their website. Implemented using EventBrite API.

  • Mailing Script

    Sending mail to few people does not take that much time. However when the number of contacts are large it might turn out to be tedious.

    Therefore this is a project which sends mails to multiple contacts in seconds of time.

    I implemented this using smtplib module in python. The smtplib module defines an SMTP client session object that can be used to send mail to any Internet machine with an SMTP or ESMTP listener daemon.

  • Budget App

    This is one of my earliest projects. The app keeps a record of your monthly budget and the expenditure for different things bought. It therefore provides information about the money left and alerts when no money is left to spend.

    This is implemented using pure Javascript. The data is stored using the javascript's localstorage facility.

  • Loco-Weather

    A console driven script which is used to fetch the geographical location of a place. The location can be obtained by entering the place name or the postal code.

    Further the script uses the latitude and logitude to fetch the current weather information about the place. The project is created in NodeJs by the implmentation of two APIs: LocationIQ and OpenWeatherMap API.

  • Get-Recipe

    A project which uses an API to get the recipe of any drink by it's name, ingredient, category, etc. The selected drinks can be added to the storage. Google Chrome's local storage is being used for storing the selected drinks.

    Implemented Google Custom Search API to search for further information about the drink.
