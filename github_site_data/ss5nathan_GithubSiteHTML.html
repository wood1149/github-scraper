Instagram Automation

Safely and automatically growing credibility for real businesses.

Chrome Extension

Other personal projects such as a Chrome extension which analyzes and adds to sports-betting sites.

For this project, I worked in tandem with a fellow OSU student to create a userful Chome extension. The main focus was to scrape and compare data between different sportsbetting sites. Data on from different sites were logged to a Node backend and ultimately large discrepncies were emailed to the user of the extension to act on.

Data is collected from multiple sites to ultimately make real-time recommendations to the user, accurate to the second.
